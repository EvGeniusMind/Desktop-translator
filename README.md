# Desktop-translator
Прозрачный оверлей захватывает область экрана, распознает текст, переводит и показывает перевод поверх оригинала в реальном времени. В настройках задаются горячие клавиши, режим захвата (весь экран, окно, прямоугольник), стиль перевода и движок. Приложение кэширует результаты, минимизируя задержки.
